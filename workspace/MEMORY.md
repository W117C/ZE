## 📚 经验教训

### 分析报告
1. 黄金不是低收益资产，而是周期性对冲工具（长期年化6-8%）
2. 1周涨跌不能用来定性资产类别（周期错配）
3. 主动基金≠ETF，单日对比没有意义
4. 阶段性牛市数据不可外推（股票型长期年化8-15%）

### 系统配置
1. Gateway service 配置路径不一致会导致模型切换失效
2. 配置文件改了必须重启 Gateway 才能生效
3. 重启后需验证实际运行模型，不能只看配置文件

### 执行纪律
1. 复杂任务必须先写计划文件再执行
2. 文件产物必须立即用 message 发送给用户
3. 用户说"不要上传等我命令"时，必须严格遵守
