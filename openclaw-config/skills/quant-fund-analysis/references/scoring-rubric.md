# Scoring Rubric (评分标准)

## 收益质量 (20%)
| 年化收益 | 分数 |
|---------|------|
| ≥15% | 9-10 |
| 10-15% | 7-8 |
| 5-10% | 5-6 |
| <5% | ≤4 |

## 回撤控制 (20%)
| 最大回撤 | 分数 |
|---------|------|
| <10% | 9-10 |
| 10-20% | 7-8 |
| 20-30% | 5-6 |
| >30% | ≤4 |

## 稳定性 (20%)
- Sharpe ≥1.5 + Calmar ≥2.0: 9-10
- Sharpe ≥1.0: 7-8
- Sharpe 0.5-1.0: 5-6
- Sharpe <0.5: ≤4

## 可解释性 (15%)
- 收益来源清晰: 9-10
- 策略类型明确: 7-8
- 部分依赖黑箱: 6-7
- 讲不清: ≤5

## 容量 (10%)
| 规模(亿) | 分数 |
|---------|------|
| ≥1000 | 9 |
| 500-1000 | 8 |
| 100-500 | 7 |
| 50-100 | 6 |
| <50 | ≤4 |

## 管理人 (10%)
| 实盘年数 | 分数 |
|---------|------|
| ≥5年 | 9 |
| 3-5年 | 7-8 |
| 1-3年 | 6 |
| <1年 | ≤4 |

## 费用 (5%)
| 费率(%) | 分数 |
|---------|------|
| ≤1.0 | 9 |
| 1.0-1.5 | 7-8 |
| 1.5-2.0 | 6 |
| >2.0 | ≤4 |

## 评级
| 等级 | 标准 |
|------|------|
| A | ≥8.5 |
| B | 7.0-8.5 |
| C | <7.0 |
