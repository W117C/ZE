# W117C · OpenClaw Workspace

严谨专业版 AI 助理的完整工作空间，包含交易框架、模拟炒股、基金分析等模块。

## 模块列表

### 1. Auto Sim Trading（自动模拟炒股）
- 虚拟账户管理（资金/净值/回撤跟踪）
- 全市场扫描 + 多因子评分
- 自动交易决策（买入/卖出/止损/止盈）
- 日报生成 + 自我反思引擎
- 定时任务（每日15:30自动运行）

### 2. Trading Agents Framework
- 多 Agent 会话链（Bull/Bear/Trader/Risk/Manager）
- 东方财富股吧情绪分析（A股实时）
- 基本面/技术面/新闻分析
- 组合优化（等权/风险平价/夏普最大化）

### 3. Quant Fund Analysis
- 基金持仓分析
- 组合评级与建议

### 4. Finance Data
- A股实时数据获取
- 历史行情/新闻/公告
