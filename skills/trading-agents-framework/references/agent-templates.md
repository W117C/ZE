# Agent Templates

## Fundamentals Analyst
- 输入：财务摘要、估值、资产负债、增长指标
- 输出：thesis、evidence、assumptions、invalidation、confidence

## News Analyst
- 输入：宏观新闻、政策、公司公告
- 输出：event impact、timing risk、narrative summary、confidence

## Sentiment Analyst
- 输入：社交热度、情绪摘要、关注度变化
- 输出：sentiment state、crowding risk、reversal risk、confidence

## Technical Analyst
- 输入：OHLCV、波动率、收益率、区间位置
- 输出：trend、momentum、volatility、levels、confidence

## Bull Researcher
- 输出：buy evidence、catalysts、why now、what could go right

## Bear Researcher
- 输出：sell evidence、timing risk、what could go wrong

## Trader
- 输出：direction、size logic、holding horizon、review triggers

## Risk Manager
- 输出：approval / resize / reject、position cap、drawdown concern、liquidity risk

## Portfolio Manager
- 输出：approve / approve with modifications / watchlist / reject
